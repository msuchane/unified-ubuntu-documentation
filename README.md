This is a documentation repository for the Ubuntu Desktop and Ubuntu Server products. [View the OS documentation](https://documentation.ubuntu.com/ubuntu-os/latest/) this repository produces.

The documentation is written in MyST-flavored Markdown and built with the Sphinx and Read the Docs frameworks. It uses the [Canonical Sphinx Stack](https://github.com/canonical/sphinx-stack) extension. [View the Sphinx Stack documentation](https://documentation.ubuntu.com/sphinx-stack/).
